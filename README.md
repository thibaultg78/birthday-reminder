# Rappels pour les anniversaire de la famille

- Envoi d'email et de SMS avec Gmail et Twilio
- Fonctionne avec la page Google Sheet de mon compte GMail (Anniversaires Famille)
- Il y a un projet associé dans GCP avec un accès Google Sheet API. La page Google Sheet a été partagée avec l'adresse email de Google Sheets API
- Envoie d'un email 7 jours avant chaque date
- Envoie d'un email le 1er du mois avec tous les anniversaires du mois
- Envoie d'un SMS le jour J avec Twilio
